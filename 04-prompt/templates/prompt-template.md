# 提示模板（基础框架）

## 目的

用于创建新提示时的起始框架。将此文件复制到对应子目录，按实际任务填写各字段。

## 输入

- `{{任务描述}}`：要完成的具体任务
- `{{上下文}}`：相关代码、接口定义或背景信息
- `{{约束}}`：不能做什么、必须兼容什么
- `{{验收标准}}`：如何判断输出合格

## 提示

```
你是一名经验丰富的工程师。请完成以下任务：

任务：{{任务描述}}

上下文：
{{上下文}}

约束：
{{约束}}

验收标准：
{{验收标准}}

请先简述你的思路，再给出实现。如果任务不够明确，先提问而非直接猜测。
```

## 预期输出

- 简短的思路说明（2-3 句）
- 满足验收标准的实现内容
- 如有不确定项，列出假设

## 备注

- 版本：v1.0
- 优化历史：初始模板
- 已知限制：通用模板效果不如针对具体场景优化的提示；在使用中逐步特化
