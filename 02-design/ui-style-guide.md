### 字体栈

```
font-family: [待填写，如：-apple-system, BlinkMacSystemFont, "Inter", sans-serif]
```

### 尺寸阶梯

| Token | 尺寸 | 行高 | 字重 | 用途 |
|-------|------|------|------|------|
| `text-xs` | [待填写，如：12px] | [待填写] | 400 | 标签、角标 |
| `text-sm` | [待填写，如：13px] | [待填写] | 400 | 辅助说明 |
| `text-base` | [待填写，如：14px] | [待填写] | 400 | 正文 |
| `text-md` | [待填写，如：16px] | [待填写] | 500 | 小标题 |
| `text-lg` | [待填写，如：20px] | [待填写] | 600 | 模块标题 |
| `text-xl` | [待填写，如：24px] | [待填写] | 700 | 页面标题 |

### 使用规则

- 正文字重最大 500，标题可用 600/700
- 禁止使用 `text-decoration: underline` 表达层级，用字重区分
- 字体颜色只能使用 `color-text-*` Token

## 间距系统

基准单位：`[待填写，如：4px]`

| Token | 值 | 典型用途 |
|-------|----|---------|
| `space-1` | [如：4px] | 图标与文字间距 |
| `space-2` | [如：8px] | 列表项内边距 |
| `space-3` | [如：12px] | 表单元素内边距 |
| `space-4` | [如：16px] | 卡片内边距 |
| `space-6` | [如：24px] | 区块间距 |
| `space-8` | [如：32px] | 模块间距 |
| `space-12` | [如：48px] | 页面级留白 |

**规则**：所有间距必须是基准单位的整数倍，禁止使用奇数 px 值。
